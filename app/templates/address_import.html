Adressbuch – Import (CSV)

Import ersetzt immer komplett alle Adressen im Standard-Adressbuch. Kein Überschreiben/Merge.
Tastatur: Esc zurück · X Export (Vorlage)
Zurück Export (CSV)
Achtung: Replace-All
- Die vorhandenen Adressen werden vor dem Import gelöscht und anschließend neu importiert.
- Trennzeichen: ; · Zeichensatz: UTF-8 · Header: DB-Feldnamen
- Pflichtspalten: nachname, vorname, wohnort
- Import ist gesperrt, wenn es bereits Turnier-Teilnehmerdaten gibt.
Tipp: Nutze zuerst Export, bearbeite die Datei, importiere sie dann wieder.
